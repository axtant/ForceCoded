# Sum of Two Numbers

Given two integers, print their sum.

## Input Format

A single line with two space-separated integers a and b.

## Output Format

A single integer — the sum of a and b.

## Constraints

- -10^9 ≤ a, b ≤ 10^9

## Examples

**Input:**
```
3 5
```

**Output:**
```
8
```

**Input:**
```
10 20
```

**Output:**
```
30
```
